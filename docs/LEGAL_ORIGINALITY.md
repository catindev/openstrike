# Legal and originality guardrails

Openstrike repository content must be original to this project.

Allowed: source code written for Openstrike, documentation, tests, generated placeholders, and fake example configs.

Not allowed: third-party game data, extracted media, local private paths, copied restricted source, official logos, screenshots, or media samples.

Implementation rule: use references to understand behavior and formats, then write original Godot code. Mark uncertain facts as `TO VERIFY`.
